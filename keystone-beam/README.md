# Keystone & Beam — Structural Reinforcement Specialists

This repository contains a minimalist, single‑page marketing website for a hypothetical general contractor called **Keystone & Beam**.  The site is designed to convey reliability, craftsmanship and a love of historic homes while remaining modern and easy to navigate.  All assets are self‑contained so it can be deployed to a static host (Netlify, Vercel, GitHub Pages) without any additional build steps.

## Getting Started

To preview the site locally just open `index.html` in a modern browser.  Since the page is static there is no build process required.

### Deploying

1. Upload the contents of the `keystone-beam-website` directory to your preferred static hosting service (e.g. [Netlify](https://www.netlify.com/), [Vercel](https://vercel.com/)).
2. Ensure that `index.html` is served at the root of your domain and that all assets (`css`, `js`, `images`, `assets`) are preserved relative to it.

### Customizing the Brand

The brand name and tagline appear in the header, hero and footer.  To update them:

* Open `index.html` and search for `Keystone &amp; Beam`—replace this text with your company name.
* Update the tagline in the hero `<h1>`/`<h2>` elements and in the footer.
* The logomark is stored in `assets/logo.svg`.  The provided logo shows a keystone shape above a horizontal beam.  Feel free to swap this SVG for your own artwork; just keep the filename and dimensions (64×64 viewBox) so the layout remains consistent.

All of these images were generated using AI and are royalty‑free.  To use your own photography instead:

1. Place your new images into the `/images` folder.  Use modern formats such as **WebP** or **AVIF** for best performance.  For compatibility you may also include JPEG/PNG fallbacks.
2. Update the `<img>` references in `index.html` to point to the new filenames.
3. Adjust the `height`/`object-fit` properties in `css/styles.css` if your images require different cropping.

### Contact Form Integration

The contact form in the *Get In Touch* section posts to [Formspree](https://formspree.io) by default.  To activate the form:

1. Create a new form on Formspree and obtain your form ID.
2. Replace `https://formspree.io/f/{replace-with-your-code}` in `index.html` with your actual endpoint (e.g. `https://formspree.io/f/xyzabcde`).
3. When visitors submit the form you will receive an email with their message.  Alternatively, you can configure Netlify Forms by changing the `action` to `"#"` and adding `data-netlify="true"` to the `<form>` element.

### Accessibility & Performance

The site follows modern accessibility practices:

* Semantic HTML5 structure with headings in order.
* ARIA attributes on interactive controls (navigation toggle, lightbox, carousel).
* Focus outlines are visible for keyboard users.
* Images include descriptive `alt` text and do not convey information purely through colour.
* Colour contrast meets WCAG AA guidelines.
* JavaScript is unobtrusive and functionality degrades gracefully if disabled.

Performance best practices include:

* Modern image formats and lazy‑loading (thanks to CSS `object-fit` and viewport cropping).
* Minified CSS and JavaScript (hand‑written and succinct).
* Sticky navigation bar reuses GPU acceleration for smooth scrolling.
* The hero image is served as a background to allow the page to render immediately while the rest of the content loads.
